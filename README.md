# libroshare

***libroshare*** — is a web-based book sharing platform that allows users to discover, share, and exchange books within a community. Users can browse available books, add their own books for sharing, request books from others, and manage their personal library.

The platform focuses on peer-to-peer book exchange, encouraging reading culture while reducing unnecessary purchases.
